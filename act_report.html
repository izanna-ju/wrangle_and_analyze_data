Insight 2. How does the most like images affects the number of retweets¶

As can be seen on the scatter plot below. There is a strong positive correlation(approximately 0.91) between the number of likes a dog image accumulate and the number of times the image was retweeted. This makes sense because as the number of likes on an image increase on social media platform the greater the possibility of that image been retweeted.
